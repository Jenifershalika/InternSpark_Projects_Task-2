# InternSpark_Projects_Task-2

Sales Performance Analysis

A comprehensive analysis of sales data to understand trends, customer behavior, and product performance. This project demonstrates data cleaning, aggregation, visualization, and insights generation using Python and Power BI.

📂 Project Overview

The goal of this project is to analyze historical sales data to:

Identify top-performing products and categories

Understand customer purchasing patterns

Track sales trends over time

Evaluate shipping performance and delivery efficiency

Highlight opportunities to optimize revenue

🛠️ Tools & Technologies

Python: Pandas, NumPy, Matplotlib, Seaborn for data analysis and visualization

Power BI: Dashboard creation and interactive visualizations

Jupyter Notebook: Exploratory data analysis and reporting

📊 Dataset

The dataset contains the following columns:

Column Name	Description
Customer ID	Unique identifier for each customer
Order ID	Unique identifier for each order
Order Date	Date of purchase
Ship Date	Date of delivery
Ship Mode	Shipping method
Product Category	Category of product purchased
Product Price	Unit price of product
Quantity	Quantity purchased
Total Purchase Amount	Total cost of each order
Returns	Indicator if product was returned
Customer Age	Age of the customer
Gender	Customer gender
Churn	Indicator if customer churned

Data is cleaned and preprocessed for accurate analysis.

🔍 Analysis Performed

Data Cleaning

Handling missing values and duplicates

Correcting data types

Feature Engineering

Calculating total purchase amount

Deriving customer segments and purchase frequency

Exploratory Data Analysis (EDA)

Sales trends over time

Top-selling products and categories

Customer segmentation by age, gender, and churn risk

Return patterns and their impact on revenue

Visualization & Dashboards

Interactive Power BI dashboards showing key metrics

Charts for product performance, customer behavior, and shipping efficiency

📈 Key Insights

Certain product categories consistently generate the highest revenue

Specific customer segments (age/gender) show higher repeat purchase rates

Returns significantly impact net revenue in selected categories

Faster shipping modes correlate with higher customer satisfaction
